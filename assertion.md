## 环视
肯定逆序环视，左边是 Y
```
(?<=Y)
```
`(?<=\d)th` 左边是数字的 th，能匹配 9th

否定逆序环视，左边不是 Y
```
(?<!Y)
```
`(?<!\d)`th 左边不是数字的 th，能匹配 health

肯定顺序环视，右边是 Y
```
(?=Y)
```
`six(?=\d)` 右边是数字的 six，能匹配 six6

否定顺序环视，右边不是 Y
```
(?!Y)
```
`hi(?!\d)` 右边不是数字的 hi，能匹配 high
