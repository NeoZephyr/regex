## 分组与编号
括号在正则中可以用于分组，被括号括起来的部分会被保存成一个子组

```
(\d{4}-\d{2}-\d{2}) (\d{2}:\d{2}:\d{2})
```
```
2020-05-10 20:23:05
```

这个正则中一共有两个分组，日期是第 1 个，时间是第 2 个

有些情况下，只想用括号将某些部分看成一个整体，后续不用再用它，就没必要保存子组。这时可以在括号里面使用 ?: 不保存子组。不保存子组可以提高正则的性能

```
\d{15}(?:\d{3})?
```

```
((\d{4})-(\d{2})-(\d{2})) ((\d{2}):(\d{2}):(\d{2}))
```
```
2020-05-10 20:23:05
```
